== Description ==

Wp Posts Slider - CloudBerriez can be used to show your posts as a slider.

* **Simplest way to create any posts slider.**
* Full customizable settings for users.
* Pretty easy to use, with so much easy settings.
* Any posts, pages or custom posts can be shown into slider.
* WooCommerce's products can also be shown into slider, with effected designs.

== Installation ==

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install of Wp Posts Slider - CloudBerriez , log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "Wp Posts Slider - CloudBerriez" and click Search Plugins. Once you've found our Wp Posts Slider - CloudBerriez plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking "Install Now".

= Manual installation =

The manual installation method involves downloading our Wp Posts Slider - CloudBerriez plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
